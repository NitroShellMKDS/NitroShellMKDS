### My Views on Programming
- Programs are better when they’re open-source.
- If a project is open-sourced by its creators, it should stay free. (Aseprite is a prime example of what not to do; LibreSprite shows how it should be done.)
- Privacy matters: programs should respect your data. Bug reporting is an exception if done right; users should be asked whether they want to opt in to sharing error reports.
- Never criticize someone’s code quality, whether they’re a novice or a seasoned professional.
- Contribution is togetherness. When we collaborate on each other’s projects, we can accomplish anything.
